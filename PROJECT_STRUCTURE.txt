KEY COMPONENTS
==============

1. main.dart
   - App entry point
   - Theme configuration (light & dark)
   - Bottom navigation bar
   - Theme toggle functionality

2. mood_entry.dart (Model)
   - Data structure: id, emoji, note, date
   - JSON serialization
   - Helper methods: isPositive, isNegative

3. mood_storage.dart (Service)
   - Save mood entries
   - Load all entries
   - Delete entries
   - Clear all entries
   - Uses SharedPreferences

4. mood_log_screen.dart
   - 20 emoji selector (5x4 grid)
   - Text note input
   - Save functionality
   - Success feedback

5. mood_history_screen.dart
   - ListView.builder for efficiency
   - Card-based layout
   - Delete with confirmation
   - Pull-to-refresh
   - Empty state handling

6. analytics_screen.dart
   - Total entries count
   - Positive/Neutral/Negative statistics
   - Most common mood display
   - Empty state
